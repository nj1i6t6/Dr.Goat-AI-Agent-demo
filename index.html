現場友善體驗

解決訊號不穩與手髒操作難題

  • PWA 離線同步 斷網存手機，連線自動上傳 (IndexedDB)
  • 語音對話輸入 免打字，Web Speech API 轉文字紀錄
  • LINE 智慧整合 緊急推播與智慧通知路由

精準農務決策

從「紀錄工具」進化為「決策大腦」

  • 近親繁殖紅燈警告 配種前自動比對族譜 (Sire/Dam)
  • 生長曲線 PR 值 取代不穩定的 ML 預測，改用統計標準
  • FCR 換肉率與 ESG 整合飼料庫存計算真實飼養效益

企業級架構

支援大型契作與多場域管理

  • 多層級權限 (RBAC) 場主/員工/獸醫 角色分離與權限微調
  • 完整場域層級 農場 → 區 → 舍 → 欄 (Pen) 結構化
  • 混合雲 RAG 架構 本地與 Gemini File Search 雙模組切換
